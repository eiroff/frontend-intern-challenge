# Desafio estagiário: HTML/CSS/JS

## Pré-requisitos
- npm
- gulp

## Instruções

- Clonar o projeto;
- Checkout no branch maria-lucia-eiroff;
- cd project;
- gulp watch;
- abrir http://localhost:3000.
